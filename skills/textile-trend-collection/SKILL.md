### 文章提取技巧
texindex文章页在未登录状态下可提取正文（内容从侧边栏区域溢出），使用以下模式按优先级尝试：

**⚠️ 重要发现（2026-04-14验证）：**
- texindex文章页（如 `/Articles/2026-4-13/649420.html`）是**动态JS渲染页面**，ScrapingBee普通模式只返回导航HTML框架，无实际正文
- `render_js=true` 参数会立即返回空（ScrapingBee对此类页面无效）
- **解决方案**：从texindex**首页**正则提取文章URL列表（`/Articles/2026-4-1[34]/[^"]+\.html`），首页内容本身可访问，且侧边栏会溢出部分文章摘要信息
- **价格数据获取**：首页已有当天主要商品报价数据（涤纶短纤/PTA/涤纶纱等），无需逐个抓文章页

```python
patterns = [
    r'<div[^>]*class="[^"]*TRS_Editor[^"]*"[^>]*>(.*?)</div>',
    r'<div[^>]*class="[^"]*article[^"]*"[^>]*>(.*?)</div>',
    r'<div[^>]*class="[^"]*content[^"]*"[^>]*>(.*?)</div>',
]
```

### 执行环境注意

**ScrapingBee调用方式**：必须用`terminal()`调用curl，**不能用Python urllib**。cron/sandbox环境中Python urllib请求ScrapingBee会报400错误，但curl请求完全正常。正确写法：
```bash
curl -s --max-time 30 \
  "https://app.scrapingbee.com/api/v1/?api_key=$SCRAPINGBEE_API_KEY&url=$(python3 -c "import urllib.parse; print(urllib.parse.quote('URL'))")&country_code=CN"
```

**双源Fallback设计**：ScrapingBee成功率 < 50%时自动降级到百度千帆AI搜索。百度千帆API：
- Endpoint: `POST https://qianfan.baidubce.com/v2/ai_search/web_search`

**采集量原则**：够写当天的文章数量即可，不要贪多。8篇文章 → 4个关键词 × 2篇 = 8条素材/天足够。

### 布行视角：五大分析角度

标注格式：`*[布行-价格传导] 涤纶短纤上调200元，绍兴柯桥坯布商跟涨5%，广州中大贸易商观望*`

## Python采集模板

```python
import urllib.request, re, time

def fetch(url):
    req = urllib.request.Request(url, headers={
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/121.0.0.0 Safari/537.36',
        'Accept-Language': 'zh-CN,zh;q=0.9,en;q=0.8',
        'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8',
    })
    try:
        with urllib.request.urlopen(req, timeout=20) as r:
            return r.read().decode('utf-8', errors='replace')
    except Exception as e:
        return f"ERROR: {e}"

def extract_article_body(html):
    if not html or html.startswith("ERROR") or len(html) < 1000:
        return None
    html = re.sub(r'<script[^>]*>.*?</script>', '', html, flags=re.DOTALL)
    html = re.sub(r'<style[^>]*>.*?</style>', '', html, flags=re.DOTALL)
    for pattern in [
        r'<div[^>]*class="[^"]*TRS_Editor[^"]*"[^>]*>(.*?)</div>',
        r'<div[^>]*class="[^"]*article[^"]*"[^>]*>(.*?)</div>',
        r'<div[^>]*class="[^"]*content[^"]*"[^>]*>(.*?)</div>',
    ]:
        m = re.search(pattern, html, re.DOTALL)
        if m:
            text = re.sub(r'<[^>]+>', ' ', m.group(1))
            text = re.sub(r'\s+', ' ', text).strip()
            if len(text) > 100:
                return text
    return None

# 使用示例
html = fetch("https://www.texindex.com.cn/Articles/2026-4-13/649419.html")
body = extract_article_body(html)
```
